Canada Post Redesign

Roles : UX/UI Designer

Duration : 4 Weeks

Overview

Canada Post serves as the foremost postal service provider in the country, delivering dependable mail and parcel services nationwide. While the current website exhibits a robust User Interface Design and Information Structure for handling intricate postal services, there are areas for enhancement. During the onboarding process, the initial fold lacks prominent visibility of frequently sought-after services, and the navigation system can be overwhelming, posing challenges for users in locating specific services. How might we optimise the user experience by implementing a streamlined navigation approach, ensuring intuitive access to online postal services?
The Current Iteration of the Website

Problem

  • Hero Page lacks prominent visibility of sought-after services.
  • Four functions stacked in one corner make it feel crowded.
  • Navigation system can be overwhelming.

Proposed Solutions

  • Priority the content of homepage
  • Provide a clear Information Architecture

Research

My research initially focused on the user tests conducted on the current website. This approach helped me understand how users interact with the website and identify specific issues that need to be addressed. Secondly, observe the postal peers to understand how other postal websites are addressing similar issues.

What I found

  • Four out of five people interact with the most common task section rather than the navigation bar.
  • The absence of hover-triggered expansion in the navigation bar led to users not clicking on it.
  • Top Four reason people visited postal website is check the rate, shipping online, find a postal code and find a post office.

Reflection

While the current website demonstrates a robust User Interface Design and Information Structure for handling intricate postal services, my initial curiosity focused on identifying areas for improvement. However, after conducting the user test, I found it to be quite helpful. It provided insights into user behaviour, enabling me to define the problems that need solving.
